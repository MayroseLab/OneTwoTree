
This Directory can be downloaded at the end of each run of OTT from the site:
Under Results you can find the 'Download all output' button that will download 
a sipped summary_dir of your OTT job.

The main files in this directory:
-----------------------------------------------------------------------------
1. Phylogeny (Newick format): Result_Tree_1524727033
2. MSA file (fasta format): 1524727033-concat-aligned
3. All clusters data (csv format): All_Clusters_data
4. Accessions matrix according to cluster distribution (csv format): AccessionsMatrix
5. Taxa list included in the final tree (txt file): FinalSpeciesList / Final_TaxId_vs_Name
